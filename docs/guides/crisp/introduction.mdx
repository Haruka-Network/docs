# 产品介绍

全面提升您与客户沟通效率的智能助手。

Crisp 机器人是由 Haruka Network 提供的云服务，旨在为用户提供高效的沟通体验。

它支持多种功能，适合*个人用户*和*团队使用*。

## 主要特点

- **个性化设置**：自定义回复头像与名称，提升品牌形象。
- **项目协作管理**：支持多成员项目协作，提升团队工作效率。
- **智能与人工回复切换**：人工回复后自动关闭智能回复，确保用户得到优质服务。
- **智能关键词匹配**：基于自定义关键词和正则表达式触发逻辑，精准响应用户需求。
- **基于 OpenAI 的智能回复**：利用先进的 AI 技术，提供智能化的互动体验。
- **选项回复**：结合关键词，提供多样化的回复选项，满足不同用户需求。
- **Telegram 推送**：实时推送至 Telegram 话题群，将不同的聊天会话分栏展示。
- **Telegram 回复**：支持文字以及图片回复内容，并实时推送给客户。
