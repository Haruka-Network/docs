# 智能回复

在收到 Crisp 消息后，系统将首先匹配关键词回复，然后再进行智能 AI 回复。

**请注意：智能回复将消耗您的余额，费用根据套餐的不同而异。**  
**综合费用大约为每条文字回复：￥ 0.002 左右，价格十分便宜。**
