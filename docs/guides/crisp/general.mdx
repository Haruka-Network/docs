# 基础设置

通过基础设置，您可以快速对接 Crisp 平台和 Telegram 平台。

## 安装 Crisp 插件

点击“安装”按钮将跳转至 Crisp 平台。请点击“继续”以许可我们访问您的 Crisp 会话。

![](/20241107-144227.png) 

我们将访问以下权限：
- 阅读对话
- 读写信息

## 是否启用项目

此步骤是项目的总开关。关闭后，项目将整体停用，不会触发任何效果，包括关键词回复、智能回复和 Telegram 推送等。

## 设置项目名称

设置与创建时相同的项目名称，以便在有多个项目时方便区分。

## 设置网址 ID

安装插件后，需要设置 Crisp 的网址 ID，否则将无法生效。您可以从 [Crisp 官网](https://app.crisp.chat/website/) 获取，格式为 `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`（8-4-4-4-12）。

## 设置回复名称

支持自定义插件回复时展示的名称，以提升您的品牌形象。

## 设置回复头像

当前不支持自行上传头像，但您可以自定义插件回复时展示的头像，以提升品牌形象。

## 输入 Telegram 群 ID

如果您希望推送消息至 Telegram 并在 Telegram 中回复，请按照以下步骤操作：

1. 创建群组，并搜索 @haruka_ai_bot，将其拉入群聊中。
2. 点击右上角管理群，将 Topics/话题设置为开启，并保存。
3. 再次点击右上角管理群，在 Administrators 中将该 bot 设置为管理员，并确保其拥有 Manage Topics 权限，保存。
4. 右键点击 General，设置为置顶（Pin）。
5. 进入该话题，输入 `/ping`，将返回群组 ID 信息。请注意，这个 ID 中包含“-”，务必一并保存。

## 人工介入切换

- **免费版**：时间限制为 1800 秒（即半小时）。
- **非免费版**：可自由设置，最短生效时间为 60 秒。

该设置的目的是在人工（Crisp 端、Telegram 端）回复介入后，关闭 AI 智能回复的时间。

## 欢迎模式

如果您希望用户访问网站时，自动推送一条欢迎消息，请将欢迎模式的脚本放入您网站 Crisp 脚本下方。

触发类型：
- **无动作**：访问后无需推送。
- **回复文字**：访问后推送指定文字。
- **回复选项**：访问后推送指定选项。
