# 选项管理

合理使用选项可以快速解决客户的常见问题，提升服务效率。

## 创建与编辑选项

### 选项名称
为选项起一个通俗易懂的名称，以便快速调用。

### 展示文字
展示文字是回复内容的具体说明，例如：
```
我们的人工客服可能无法及时回复，请善用我们提供的菜单功能以及知识库来找到您所需要的答案。

请点击下方的按钮来选择：
```

### 选项
每个选项中您可以创建最多 10 个小项，具体设置如下：

- **选项文字**：客户所看到的显示文字。
- **选项类型**：
    - **无动作**：点击后无响应。
    - **回复文字**：点击后回复指定文字。
    - **回复选项**：点击后回复其他选项。
    - **关闭智能回复**：点击后关闭该对话的智能回复。

完成配置后，请务必保存设置以使其生效。
