# 自动回复

在收到 Crisp 消息后，我们将优先匹配以下关键词进行自动回复。

**请放心使用，关键词匹配回复不会消耗您在平台的余额。**

## 创建选项

### 触发关键词

您可以使用正则表达式或简单的匹配规则：

- `^菜单$`：仅匹配“菜单”这两个字时触发。
- `你好`：匹配任意包含“你好”的消息，例如“你好，xxxxx”或“打你好吗”。
- `在吗|在不在`：匹配包含“在吗”或“在不在”的消息，如“在吗？？？”或“你在不在”。

### 选项类型

当匹配到关键词后，可以选择触发的类型：

- **无动作**：匹配到内容后不响应消息。
- **回复文字**：匹配到内容后以文字形式响应。
- **回复选项**：匹配到内容后以选项形式响应。
- **关闭智能回复**：匹配到内容后关闭该对话的智能回复。

## 测试匹配词

关键词触发是按照顺序从上到下进行匹配的。我们提供了测试匹配词的功能，使您能够快速且直观地查看用户发送的内容是被哪条关键词触发的。触发的关键词会高亮显示。

## 保存设置

在完成所有设置并进行测试后，请务必保存设置以使其生效。
