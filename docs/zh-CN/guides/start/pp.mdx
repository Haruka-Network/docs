Haruka Network 尊重您的隐私。  
我们严格遵守《通用数据保护条例》(GDPR) 和英国《数据保护法 2018》。

### 1. 数据收集与使用

1. **收集范围**：我们收集您在注册和使用服务时提供的个人信息（如姓名、邮箱、IP地址等）。
2. **处理基础**：我们基于履行合同、合法利益或您的明确同意处理数据。
3. **用户内容**：您上传的内容（如图片、网站文件）的知识产权归您所有。通过上传，您授予我们为提供服务所需的存储、处理和分发的非排他性许可。

### 2. GDPR 用户权利

根据 GDPR，您享有以下权利：

* **访问权与更正权**：要求获取数据副本或更正不准确数据。
* **删除权**：在特定情况下要求删除您的个人数据（“被遗忘权”）。
* **限制处理与反对权**：限制或反对我们对您数据的处理。
* **数据可携带权**：要求以结构化格式接收您的数据。

### 3. 数据保留与删除

1. **保留期**：我们仅在必要期间内保留数据。
2. **终止后数据**：服务终止后的 **30天内**，您可以要求检索数据。超过此期限，我们可能会永久删除您的数据。
3. **数据传输**：如需将数据传输至欧洲经济区 (EEA) 外，我们将确保采用适当的保护措施。

### 4. 隐私联系方式

如您对隐私政策有疑问或需要行使您的权利，请联系：`support@haruka.cloud`
