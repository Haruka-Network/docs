本政策规定了 Haruka Network 服务允许和禁止的用途。使用我们的服务即表示您同意遵守以下规定。

### 1. 禁止的内容 (全平台通用)

您不得利用我们的服务存储、发布或传输以下内容：

* **非法内容**：违反任何适用法律法规的内容，包括宣扬犯罪、破坏商标/版权/专利保护的内容。
* **恶意内容**：包含病毒、木马、勒索软件或其他可能损害服务器或访客计算机的程序。
* **侵权内容**：未经授权的盗版软件、破解程序、受版权保护的视频或音频文件。
* **色情/成人内容**：任何形式的成人、色情或淫秽内容。
* **垃圾邮件**：发送垃圾邮件（Spam）或未经请求的广告邮件。

### 2. 特定产品使用限制

#### A. 虚拟主机 (Virtual Hosting)

* **禁止网络代理**：严禁用于网络代理目的（包括 zmirror 及类似代理网站）。
* **资源使用**：若您使用的系列没有专用资源标识，请合理使用资源，不得干扰其他客户的正常使用。
* **文件限制**：Haruka Network 保留不经通知删除盗版、破解、色情及版权音视频文件的权利。

#### B. 虚拟专用服务器 (VPS)

* **禁止挖矿**：严禁进行任何形式的加密货币挖掘活动。
* **禁止公开代理**：禁止提供免费的公共代理服务。一经发现，账户将被暂停至过期。
* **禁止黑客行为**：严禁进行IP扫描、端口扫描、对外攻击、试图入侵其他网络或计算机的行为。
* **高资源占用**：禁止放置聊天室、大型下载站或其他严重占用资源的程序（包括但不限于无节制的爬虫）。

#### C. CDN 图片存储与 Crisp 插件

* **合规性**：上传的内容必须符合知识产权、隐私权和名誉权规定。
* **禁止滥用**：不得干扰服务运行，不得上传非法、威胁、骚扰或包含恶意软件的文件。

### 3. 违规处理

* **审查权利**：我们保留审查、监控、编辑或删除任何内容的权利（但无义务）。
* **处理措施**：任何违规行为可能导致服务暂停、终止或删除。
* **法律报告**：对于非法活动，我们保留向执法机构报告并配合调查的权利。
